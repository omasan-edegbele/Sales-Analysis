## Overview
This project analyzes fourth-quarter sales data for **AAL**, an Australian clothing brand operating across multiple states and customer demographics.  
The goal is to extract actionable insights that support sales strategy, regional investment decisions, and marketing optimization.

This analysis is presented as a personal portfolio project demonstrating end-to-end data analysis and visualization skills.

---

## Business Objective
- Identify Australian states generating the highest sales revenue
- Detect underperforming regions and recommend targeted sales programs
- Analyze sales trends across customer groups (Kids, Women, Men, Seniors)
- Identify peak and off-peak sales periods to inform marketing strategy

---

## Dataset
- **Source:** CSV file (`data/AusApparalSales4thQrt2020.csv`)
- **Period:** Q4 2020
- **Scope:** State-wise, demographic-wise, and time-based sales data

---

## Approach
- **Data Wrangling:** cleaned missing and invalid entries, normalized numerical features, and aggregated data using `groupby()`
- **Analysis:** descriptive statistics on sales and units sold, including mean, median, and standard deviation
- **Visualization:** state-wise and demographic-wise dashboards, time-based sales trends
- **Reporting:** documented insights and recommendations in JupyterLab using Markdown

---

## Tools Used
- Python (pandas, numpy, scipy)
- seaborn, matplotlib
- JupyterLab

---

## Key Takeaways
- Identified top-performing and underperforming states
- Revealed demographic-based sales patterns
- Highlighted time periods with the highest sales activity
